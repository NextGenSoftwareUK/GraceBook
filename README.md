# GraceBook

**The Conscious Social Network · OASIS Omniverse**

GraceBook is the Web4 social network of the OASIS Omniverse — a gold/teal-themed conscious alternative to traditional social media where kindness, authenticity and positive contribution earn karma rather than likes. Built on the OASIS universal avatar system, every post, connection and act of generosity is recorded on-chain.

## What is GraceBook?

- **Conscious posting** — share thoughts, art, wisdom and positivity; earn karma for quality over quantity
- **Universal avatar profiles** — your OASIS avatar is your GraceBook identity; karma earned here carries everywhere
- **Graceful connections** — connect with people through shared values and interests, not follower counts
- **Karma feed** — a feed ranked by positive impact, not engagement bait
- **Integration** — linked to all OASIS apps (JLA, GHN, Noah's Ark ARN, Our World, OPORTAL)

## Color Scheme

Gold (`#f5c518`), Teal (`#00d4aa`), Rose (`#ff5fa0`) on deep space background (`#030714`).

## Tech Stack

| Layer | Detail |
|---|---|
| Front-end | Single-file `index.html` — inline CSS + vanilla JS |
| Fonts | Orbitron, Rajdhani, Share Tech Mono (Google Fonts) |
| OASIS API | `@oasisomniverse/web4-api@2.0.2` via esm.sh |
| Hosting | Static — any CDN or static host |
